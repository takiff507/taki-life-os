# Security Policy

Taki Life OS is a static local-first app. It does not use a backend, authentication system, database, analytics script, or API key.

## Reporting a Concern

If you notice a security or privacy concern, please open a GitHub issue with:

- A short description
- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser and device information if relevant

Avoid sharing private personal data in public issues.

## Supported Scope

Security review should focus on:

- Data import/export handling
- Local storage behavior
- Links and static assets
- Browser-only privacy expectations

## User Data

User data stays in the browser unless the user exports it manually. Exported JSON files may contain personal notes, task names, and expense labels, so users should store backups privately.
