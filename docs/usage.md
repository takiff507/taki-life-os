# Usage Guide

Taki Life OS is an offline dashboard, so it works without an account, database, API key, or build step. Open the app, use the tools, and your browser saves the data locally.

## Daily Workflow

1. Start on the dashboard and add anything urgent through Quick Capture.
2. Move tasks across the board as your work changes.
3. Mark habits once per day to build streaks.
4. Use the focus timer for study, coding, writing, or planning sessions.
5. Write short notes for ideas, checklists, and project logs.
6. Record simple expenses so your monthly total stays visible.

## Data Safety

The app stores data in `localStorage` on the current browser. Use the Data tab to export a JSON backup before clearing browser data or switching devices.
