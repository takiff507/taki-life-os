# Roadmap

This roadmap keeps future work focused and easy to review.

## Near Term

- Add keyboard shortcuts for common navigation and capture actions.
- Add CSV export for expenses.
- Add a first-run demo state that users can load or clear.
- Improve empty states for each view.

## Later

- Add optional task tags.
- Add weekly habit summaries.
- Add note folders or note labels.
- Add printable daily planning view.

## Quality Goals

- Keep the app offline-first.
- Avoid build tooling unless it clearly improves maintainability.
- Keep data export/import readable and stable.
- Preserve accessibility and mobile usability with every change.
