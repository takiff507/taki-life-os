# Keyboard Shortcut Plan

Keyboard shortcuts can make Taki Life OS faster for daily use. This document keeps the first shortcut set simple and predictable.

## Proposed Shortcuts

- `g d` - Go to Dashboard
- `g t` - Go to Tasks
- `g h` - Go to Habits
- `g f` - Go to Focus
- `g n` - Go to Notes
- `g e` - Go to Expenses
- `n` - Create a new item in the current view
- `/` - Focus search when available

## Rules

- Shortcuts should not interfere while typing in inputs or textareas.
- Shortcuts should have visible focus behavior.
- Destructive actions should not run from a single key press.
- Mobile should keep working without relying on shortcuts.

## Testing

Before adding shortcuts, test forms, notes, and timer controls to make sure typing remains normal.
