# Data and Privacy

Taki Life OS is designed as a local-first personal dashboard. It does not need a backend server, login, analytics service, API key, or external database.

## Where Data Lives

Data is saved in the browser's `localStorage` under the current site origin. That means the same browser and device can reopen the dashboard and keep the user's tasks, habits, notes, focus sessions, and expenses.

## What Is Not Collected

This project does not intentionally collect, upload, sell, or sync personal data. The static app runs in the browser and does not include tracking scripts.

## Backups

Use the Data tab to export a JSON backup. Keep backups somewhere private because they may include personal notes, task names, and expense labels.

## Limitations

Browser storage can be cleared by the user, privacy tools, or browser settings. Export important data regularly if the dashboard becomes part of a daily routine.
